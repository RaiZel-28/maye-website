# Personal Profile Page

Brief description: This is a simple personal profile website built with HTML, CSS, and optional JS.

How to run:
1. Open `index.html` in a browser.
2. Or serve locally: `npx serve` or `python -m http.server` then open `http://localhost:8000`.

Deployment:
- Push to GitHub and enable GitHub Pages from the repository's settings (use the main branch or gh-pages branch).
